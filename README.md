TEAKWOOD
========

Teakwood aims to help scientist using open sources tools easily.
Teakwood is a web-based deployment and visualization framework that provides mutiple functions including, 
job sumbittion, mornitoring, report generation.

USEAGE
======

How to set up teakwood to your local server?

1) create a virtual environment use virtualenv;
2) with in the virtual environment, use "pip install -r requirements.txt" to install the dependencies;
3) create an empty database "TEAKWOOD", then use "syncdb" command to prepare your data tables;
4) "runserver" command to start your business, done.
